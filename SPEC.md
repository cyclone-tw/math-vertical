# math-vertical - 數學直式計算練習工具 完整 Spec

## 專案概述
資源班學生（學習障礙/輕度知能障礙）用的數學直式計算練習網頁工具。
單一 index.html 檔案，無後端，支援 Google Apps Script 記錄成績。

## 技術棧
- 純 HTML + CSS + JavaScript（單一 index.html）
- Google Apps Script（透過 clasp 部署，用於寫入 Google Sheets）
- 無框架，無依賴

## 介面結構（三個畫面切換）
[老師設定畫面] → 開始練習 → [學生練習畫面] → 結束 → [結果報告畫面]

---

## 畫面一：老師設定

### 設定項目
- 學生姓名（文字輸入）
- 運算類型：加 / 減 / 乘 / 除（可複選）
- 第一個數位數：1 / 2 / 3 位
- 第二個數位數：1 / 2 / 3 位
- 是否允許進位/退位：開關（關閉則只出不進退位的題）
- 出題數量：5 / 10 / 15 / 20 或自訂
- 限時設定：開關 + 分鐘數（關閉則純計時不限時）
- 鷹架等級：「再想想」/ 「步驟提示」/ 「完全引導」
- 作答模式：「直接輸入答案」/ 「逐步引導輸入」
- Google Sheets Script URL：文字輸入（可留空，留空只能本機匯出）

---

## 畫面二：學生練習

### 視覺設計
- 超大字體（數字最小 64px）
- 高對比配色（深藍底白字為主）
- 位值欄位顏色區分：個位=綠色框、十位=藍色框、百位=橘色框
- 進退位格子：亮黃色小框，明顯標示
- 整體乾淨清晰，減少干擾元素

### 直式排版規則（位值對位）
```
         [百] [十] [個]
              2    3
          +   1    5
         ──────────────
     [進] [百答][十答][個答]
```
每個位置是獨立 input 框，CSS grid 確保對齊，位值絕對正確。

### 作答模式 A：直接輸入
- 答案列有一個大輸入框
- 數字自動靠右對齊，視覺上位值正確
- 輸入後按 Enter 或確認鍵提交

### 作答模式 B：逐步引導
- 從個位開始 highlight，其他位置灰掉
- 學生填入個位答案 → 需進退位時進位框亮起 → 填進位 → 解鎖下一位
- 依序從右到左完成每個位
- 乘法：分步驟（個位乘 → 十位乘 → 加總）
- 除法：分步驟（試商 → 乘積 → 相減 → 拉下下一位）

### 乘法特殊設計（部分積位值概念）
以 23 × 4 為例，分三步：
1. Step 1：「先算個位 3 × 4 = ?」→ 學生填 12，系統引導寫 2 進位 1
2. Step 2：「再算十位，2 其實是 20，20 × 4 = ?」→ 「2」高亮 + 標示「= 20」，學生填 80
3. Step 3：「把 12 和 80 加起來」→ 顯示加法直式，學生填最終答案

### 鷹架提示（答錯後觸發）
- **「再想想」模式**：溫和文字提示（例：「再仔細看看個位喔！」），換提示色，可重答一次
- **「步驟提示」模式**：顯示該步驟的計算方法（例：「個位：3+5=8，寫8」），但不給答案
- **「完全引導」模式**：逐步顯示每個步驟的完整計算過程，學生照著填

### 動畫音效
- 答對：✨ 星星爆炸動畫 + 鼓勵音效（Web Audio API 合成）
- 答錯：輕微震動效果
- 全部完成：大型彩帶慶祝動畫 + 歡呼音效
- 靜音開關（右上角）
- 音效在瀏覽器靜音/無音效時降級為純視覺動畫

### 計時器
- 右上角顯示已用時間（MM:SS）
- 有設限時：最後 10 秒，計時框變紅色閃爍警示
- 時間到：自動結束，跳轉結果畫面
- 無限時：純計時，事後在結果看用時

---

## 畫面三：結果報告

### 內容
- 學生姓名 + 練習日期時間
- 運算類型 + 設定
- 總題數 / 答對數 / 正確率（大字顯示）
- 用時
- 每題明細表格：題目 / 學生答案 / 正確答案 / 對錯標示

### 匯出功能
- **下載 CSV 按鈕**：產生 CSV 檔，瀏覽器下載
- **複製格式按鈕**：複製 tab 分隔格式，可直接貼進 Google Sheets
- **上傳 Sheets 按鈕**（設定了 Script URL 才顯示）：POST 到 Apps Script

### CSV 格式
```
姓名,日期時間,運算,第一數位數,第二數位數,進退位,題目,學生答案,正確答案,結果,用時(秒)
小明,2026-03-10 16:30,加法,2位,1位,允許,23+5,28,28,正確,8
```

- 再練一次（同設定）/ 重新設定 按鈕

## Google Apps Script (gas/ 目錄)

### 檔案結構
```
gas/
  Code.gs          ← 主邏輯
  appsscript.json  ← 設定
  .clasp.json      ← clasp 設定（部署後產生）
```

### Code.gs 功能
- doPost(e)：接收 JSON，寫入 Sheets
- 第一次執行時自動建立標題列
- 支援 CORS（任何來源都可 POST）

### 寫入欄位
姓名、日期時間、運算類型、位數設定、進退位設定、題目、學生答案、正確答案、結果、用時(秒)

---

## 出題引擎規則

### 加法
- 控制位數：根據設定的位數範圍隨機產生
- 不進位：確保每位相加 < 10
- 允許進位：正常隨機，可跨位進位

### 減法
- 確保被減數 >= 減數（結果非負）
- 不退位：確保每位被減數 >= 該位減數
- 允許退位：正常隨機

### 乘法
- 根據位數設定（例：2位數 × 1位數）
- 不進位：確保每步乘積 < 10（限制乘數和被乘數組合）

### 除法
- 確保整除（無餘數）
- 根據商的位數控制難度
- 不退位：限制除法步驟中不出現試商困難的情況
